2.目标市场分析
	近几年，我国花卉种植的产值基本以30%以上的增速增长。中国报告大厅研究数据显示，2011年，我国花卉的种植实现总产值939.91亿元，同比增长33.68%，预计到2018年我国花卉市场销售额将超过2000亿元。 
　　随着人民生活水平的提高，花卉消费主体已由集团消费转向个人消费；同时，全国花卉消费额以年均10%以上的速度在递增，拥有13亿人口的中国无疑是一个潜力巨大的消费市场。
	有多少人养花？根据初步统计，大约有65%的人会在办公室或者居住场所养一盆以上的花卉或绿植。


三、需求分析
1.用户场景
	很多白领都会在办公室里养花，但是经常会浇水太频繁导致花草长得不好。
	很多注重生活质量的人都会在家中养花草，但是可能由于工作繁忙，或者没有经验，导致花草长的并不好。
	有些人喜欢养多肉植物，当养了很多植物时，会希望能够和同样喜欢花的朋友分享。
2.目标用户
	所有喜爱花草的人
3.用户核心价值
	解决养花草过程中遇到的所有问题。
		如不知道何时浇水施肥，或者忘记了。
		如遇到病虫害不知道该如何解决。
		
4.产品核心价值
产品定位：
希望通过本app帮助所有喜爱花草的人，都能够养好自己的花草。
希望用户通过本app结识更多拥有同样爱好的朋友。

5.功能列表
	1.核心功能
		本app的核心业务是帮助用户管理好自己的植物，定期提醒用户进行浇水、喷雾和施肥等事情。
	2.社交功能
		分享功能，用户可以将自己的养护经验分享给大家，或者将自己的植物展示给别人。
		可以加好友，可以访问好友的花园查看好友养护的植物，可以发消息进行联系
		。。。
	3.线上交易平台：广告或电商
	4.运营线下活动，以花会友。
